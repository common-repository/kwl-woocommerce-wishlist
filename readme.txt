Simple Product Wishlist for WooCommerce.

== Description ==

KWL WooCommerce Wishlist is basically unofficial addon for WooCommerce. Logged in users and Guests can both easily manage their wishlists.

= Docs & Support =

Complete Installation and Usage steps can be found on [plugin website](http://www.krstic.in.rs/kwl-woocommerce-wishlist/) and for support you can check the [support forum](https://wordpress.org/support/plugin/kwl-woocommerce-wishlist/) on WordPress.org.


= Privacy Notices =

With the default configuration, this plugin, in itself, does not:

* track users by stealth;
* write any user personal data to the database;
* send any data to external servers;

With the default configuration, this plugin, in itself, use cookies.


= Required Plugins =

* WooCommerce

= Translations =

* Translations are currently not supported.

== Installation ==

1. Upload the entire 'kwl-woocommerce-wishlist' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Refresh permalinks by click on "Save Permalinks" button in Settings > Permalinks

You will find 'Woo Wishlist' menu in your WordPress admin panel. There you can adjust plugin options.
